## 📋 Description

Calcul et analyse d'Indicateurs de Business Activité (IBA) à l'aide de SAS et R. Ce projet consiste à exploiter des données d'entreprise pour produire des indicateurs clés de performance (KPI), des tableaux de bord statistiques et des analyses de l'activité commerciale.

## 🎯 Objectifs pédagogiques
- Calculer des KPI à partir de données d'activité (CA, taux de croissance, parts de marché)
- Utiliser SAS pour la manipulation de données et les analyses statistiques
- Créer des tableaux de bord synthétiques avec R
- Interpréter les indicateurs dans un contexte économique

## 🔧 Technologies

| Outil | Rôle |
|-------|------|
| SAS Base / SAS Studio | Manipulation DATA step, PROC statistiques |
| R | Visualisations et analyses complémentaires |
| Excel | Données sources et mise en forme |
